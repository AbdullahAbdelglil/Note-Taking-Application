# Note-Taking-Application

## Features
### 1 - Login Screen For Different Users  (Login , Register)
### 2 - Any User Can Creat , Edit , And Raed Text-Based Notes
### 3 - Notes May Be General Or Secure , Its Up To You
### 4 - You Can't Creat 2 Or More Notes With The Same Title

## ! Importatnt :
## Folder Users/DataBase >> Contains 2 Folders 
## 1 >> Users Handles >> Stores Users Hanldes In Handles.txt File
## 2 >> Notes >> Stores All Notes Titles And Passwords For Secure Notes In Individual Folders

##  Once A User Is Creadted , This Folders Will Be Created
### Users/handle/General_Notes >> Stores All General Notes 
### Users/handle/Secure_Notes >> Stores All Secure Notes
### Users/DataBase/Notes/General_Notes/handle >> Stores All General Notes Titles 
### Users/DataBase/Notes/Secure_Notes/handle >> Stores All Secure Notes Titles And Passwords
